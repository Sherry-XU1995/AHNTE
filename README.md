# AHNTP

##### pip install dhg
##### pip install torch
##### pip install scikit-learn

##### numpy == 1.24.4
##### torch == 1.13.1+cu117
##### scikit-learn == 1.3.0
##### dhg ==0.9.3
##### python == 3.8.13
